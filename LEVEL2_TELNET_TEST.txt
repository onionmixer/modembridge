==기본사항==

(1) cd build ; rm -rf *.log ; ./modembridge -c ../modembridge.conf -v 2>&1 | tee server.log 이렇게 실행하면 modembridge 의 표준출력을 server.log 로 저장해서 테스트 결과를 확인할 수 있습니다.


(2)level2 에서 테스트하게될 telnet 서버의 port 정보입니다. ip 는 "127.0.0.1" 입니다.

* Port 9091: line_mode_server
* Port 9092: char_mode_server
* Port 9093: line_mode_binary_server

port 값은 modembridge.conf 의 값을 변경해서, 프로그램을 재시작해가며 테스트합니다.(설정파일 로딩기능 검증)


(3)각 서버마다 순차적으로 다음의 테스트를 각각 별개로 진행합니다. 

==테스트내용==

test 1 : 30초간 기다려서 서버로부터 timestamp 가 제대로 수신되는지 확인(결과는 log 로 확인)
test 2 : telnet 서버에 접속해서 "abcd" "한글" "こんにちは。" 를 telnet 서버로 각각 3초 간격으로 보내서 정상적으로 echo 가 오는지를 확인(전송은 프로그램 내에서, 결과는 log 로 확인)

test2 를 위해서 level2 내에 문자를 전송하는 임시 기능을 새로 추가합니다. 나중에 주석처리해서 disable 할 수 있는 모듈 형태로 제작합니다.

테스트 결과를 정리해서 level2_telnet_test_result.txt 로 저장합니다.
